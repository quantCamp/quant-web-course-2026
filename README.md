# Летняя физико-математическая школа «Квант» — Веб-разработка на Python (2026)

Добро пожаловать в репозиторий курса **«Веб-разработка на Python (Django)»**, подготовленного для Летней физико-математической школы **«Квант»**.

* научатся работать с шаблонами, формами и административной панелью Django;
* познакомятся с принципами REST API;
* получат опыт разработки веб-приложения в команде.

# Организация разработки

В репозитории используется упрощенный Git Flow.

Основная ветка:

```text
main
```

Каждый участник создает собственную ветку:

```text
feature/ivan
feature/alicey
feature/nikita
feature/arsen
```

Работа выполняется **только** в своей ветке.

После завершения разработки каждый участник демонстрирует реализованный функционал.

# Запуск проекта

## 1. Клонирование репозитория

```bash
git clone https://github.com/IMART0/quant-web-course-2026
cd quant-web-course-2026
```

## 2. Создание виртуального окружения

### Windows

```bash
python -m venv .venv
.venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv .venv
source .venv/bin/activate
```

## 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 4. Применение миграций

```bash
python manage.py migrate
```

## 5. Создание администратора

```bash
python manage.py createsuperuser
```

## 6. Запуск сервера

```bash
python manage.py runserver
```

После запуска приложение будет доступно по адресу:

```text
http://127.0.0.1:8000/
```

# Правила работы

Перед началом работы:

1. Получите последнюю версию ветки `main`.
2. Создайте собственную ветку.
3. Выполняйте изменения только в своей ветке.
4. Делайте небольшие осмысленные коммиты.
5. Регулярно сохраняйте изменения в удаленный репозиторий.

Пример:

```bash
git checkout main
git pull

git checkout -b feature/ivan

git add .
git commit -m "Добавил страницу профиля"

git push -u origin feature/ivan-profile
```
